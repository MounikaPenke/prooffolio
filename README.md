# ProofFolio 🚀  
### Evidence-Based Skill Evaluation Platform  

ProofFolio is a full-stack platform that replaces traditional resume claims with verifiable, GitHub-backed skill evidence.  
Instead of trusting bullet points, recruiters can evaluate structured repository insights and quantified skill scores.

---

## 🔥 Problem

Resumes are self-declared.  
There is no reliable way to verify a candidate’s actual coding depth, consistency, or tech-stack expertise.

---

## 💡 Solution

ProofFolio analyzes public GitHub repositories and converts real project activity into structured skill evidence and measurable scores.

---

## ⚙️ Tech Stack

Frontend:
- React.js  
- Axios  
- CSS  

Backend:
- Node.js  
- Express.js  
- MongoDB  
- JWT Authentication  

Integration:
- GitHub REST API  

---

## 🚀 Core Features

- 🔐 JWT-based role authentication (Candidate & Recruiter)
- 📂 Automated GitHub repository fetching
- 📊 Repository analysis engine (commit patterns, tech stack detection)
- 🧠 Evidence-based skill scoring system
- 🔎 Recruiter search dashboard for structured candidate evaluation

---

## 🏗️ System Architecture

1. User signs in and links GitHub.
2. Backend fetches repositories using GitHub API.
3. Repository analysis engine extracts:
   - Languages used
   - Commit frequency
   - Project structure
4. Skill evidence score is generated.
5. Recruiters can search and evaluate profiles.

---

## 📈 Why This Matters

- Reduces hiring bias.
- Encourages proof-driven portfolios.
- Shifts hiring from claims → evidence.
